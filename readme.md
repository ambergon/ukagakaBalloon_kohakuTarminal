    1 : Communicate
    2 : Teach
    3 : Input

- marker.png
    選択肢の時に表示されるmarkdownの[- ]のようなあれ。アイコン
- cursor.cur
    hover時のマウス画像
- arrow0.png
    上マーク。テキストの上下
- arrow1.png
    下マーク。テキストの上下
